# Naufal Portfolio

Portfolio website (HTML + CSS + JavaScript) yang responsif, dengan animasi ringan dan form kontak.

## Cara menjalankan
- Buka `index.html` di browser.
- Atau pakai VS Code **Live Server**.

## Struktur project
- `index.html` : halaman utama
- `style.css` : styling + theme
- `script.js` : interaksi (menu mobile, scroll, form kontak)
- `img/` : gambar

## Theme (kuning)
Warna utama dikendalikan di `style.css` pada bagian `:root`.

## Catatan form kontak
Bagian EmailJS ada di `script.js`:
- `YOUR_PUBLIC_KEY_HERE`
- `service_YOUR_SERVICE_ID`
- `template_YOUR_TEMPLATE_ID`
